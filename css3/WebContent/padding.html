padding 요소의 내부여백

padding 방향별 지정 속성

  1. padding-top: 50px;
  2. padding-right: 50px;
  3. padding-bottom: 50px;
  4. padding-left: 50px;

padding 속성값의 갯수별 지정

  1. padding: n ; 양사방으로 통일
  2. padding: n n ; 상하와 좌우 서로 같은 값으로
  3. padding: n n n ; 상, 좌우 서로 같은 값, 하 순서로 적용
  4. padding: n n n n ; 상 우 하 좌의 순서로 각각 적용
