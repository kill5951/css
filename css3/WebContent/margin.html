margin 속성값의 갯수별 지정

  1. margin: n ; 양사방으로 통일
  2. margin: n n ; 상하와 좌우 서로 같은 값으로
  3. margin: n n n ; 상, 좌우 서로 같은 값, 하 순서로 적용
  4. margin: n n n n ; 상 우 하 좌의 순서로 각각 적용

margin: n ; 양사방 n으로 통일
