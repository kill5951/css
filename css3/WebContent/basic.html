	속성: 속성값;
	속성: 속성값;
	...
}

CSS 적용방식(CSS소스의 위치)

  1. External CSS - 외부파일형식 CSS소스를 독립된 파일"*.css"로 따로 저장하고 html문서의 head영역내에
    <link href="*.css" rel="stylesheet">
    형식으로 불러와서 연결시켜 적용한다. 사이트를 일관성, 통일성있게 구성할 수 있고 유지보수가 용이함
  2. Internal CSS - 내부에 포함된 방식 CSS소스를 head영역안에
    <style type="text/css"> ... </style>
    형식으로 적용한다.
    해당 html문서에서만 스타일이 적용된다.
  3. Inline CSS - 태그 뒤에 포함된 방식
    <tag style=" ... ">
    형식으로 적용한다.
    유지보수가 어렵고 소스가 길어지나
    외부파일을 수정할 수 없는 환경에서 (임대쇼핑몰, 블로그 등) 적용할 수 있고
    세가지 방식 중에서 적용 우선순위가 가장 높다.

CSS선택자 종류 중 대표3가지

  1. Tag선택자 - Tag그대로를 선택자로 활용
    태그가 같으면 모두 같은 스타일이 적용됨(자동적용,강제적용)
  2. ID선택자 - #ID값 형식으로 선택자로 활용
    해당 페이지 내에서 ID값을 중복해서 쓸 수 없으므로 주로 구조적인 목적의 스타일에 사용함
  3. Class선택자- .class값 형식으로 선택자로 활용
    ID와 달리 class값은 중복해서 쓸수 있으므로 주로 디자인적인 목적의 스타일에 사용함. 태그 하나에 class값은 2개 이상 동시적용도 가능

전체 선택자 =*

* {속성: 속성값; }

문서의 모든 선택자(요소)를 다 불러와서 스타일을 적용함.
 CSS초기화에 활용.
 관계선택자 등과 조합하여 적용가능.
